## Tentang raincode26

**raincode26** adalah repositori publik yang dibuat untuk berbagi kode-kode yang terinspirasi dari trend TikTok. Semua kode di sini bebas digunakan untuk keperluan pribadi maupun konten — selama digunakan dengan bijak dan tidak merugikan orang lain.

---

## Daftar Trend

### ✌️ [foto_kita_blur](./foto_kita_blur/)

Efek kamera real-time berbasis Python — tunjukkan **peace sign** ke kamera, tampilan langsung **blur**. Cocok buat konten transisi TikTok.

- Deteksi tangan otomatis via MediaPipe
- Peace sign ✌️ → blur | Tangan turun → normal
- Musik latar loop otomatis
- Support webcam lokal & kamera HP (IP Webcam / DroidCam)

→ [Lihat detail & cara pakai](./foto_kita_blur/README.md)
